## How to install and use

Before you begin, you will need to have the following tools installed on your machine: [Git](https://git-scm.com), [Node.js](https://nodejs.org/en/).

In addition, it is good to have an editor to work with the code as [VSCode](https://code.visualstudio.com/).

```bash

# Clone this repository
$ git clone https://github.com/ewertonbn/github-blog

# Access the project folder in the cmd/terminal
$ cd github-blog

# Install the dependencies using npm or yarn
$ npm install

# Create a .env file with the variables below and define your username and repository to load the content

VITE_GITHUB_USERNAME = 'username'
VITE_GITHUB_REPO = 'repo'

# Run the application in development mode
$ npm run dev

# The server will start at port:3000 - access  http://localhost:3000/

```
